## Configuration

### Interfaces

```js
interface I_LogWallConfig {
    debugLevel?: number;
    title?: string;
    windowOnly?: boolean;
    shortcutKey?: string;
    docUrl?: string;
}
```


### Config Example

```js
const LogWall = useLogWall({
    title:'useLogWall Demo', 
    shortcutKey: 'digit0', 
    windowOnly: false, 
    debugLevel:3,
    docUrl: 'https://my-doc-site.xyz/docs/',
})
```

## Combos

Simple combo callbacks can be used for simple key combos.  

:::info

More advanced key combos should be handled on keyb.key change using the 
keyb.combo return value. 

:::

| Event Key | Replaced With |
|---|---|
| CONTROLLEFT | ctrl |
| CONTROLRIGHT | ctrl |
| SHIFTLEFT | shift |
| SHIFTRIGHT | shift |
| ALTLEFT | alt |
| ALTRIGHT | alt |
| KEYA | a |
| ARROWDOWN | down |

## Internal

This hook internally uses the `useState` hook to store the array and provides 
a wrapper around common array modification methods. 

Methods in `useStateArray` do not provide access to the current state for callback 
functions.
