# Installation

<!-- `@pratiq/hooks` is a collection of react-hooks and functions that provide easy 
to use tools for developing interfaces and the common functionality behind them.
 -->



Install the package from your preferred package manager

<Tabs>

<TabItem value='yarn' label='Yarn'>

```
$ yarn add @pratiq/hooks
```

</TabItem>

<TabItem value='npm' label='NPM'>

```
$ npm i @pratiq/hooks
```

</TabItem>

</Tabs>








<br />

Import the hook or function into your application:

```jsx
import { useInput, useCountdown } from '@pratiq/hooks'
```







<br />

Use them in your components like standard hooks:

```jsx
const { value, setValue, isValid, bind, ... } = useInput(config)
const { time, start, stop, reset, done, ... } = useCountdown(config)
```
