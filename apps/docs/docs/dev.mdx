## Server Side Rendering

Most hooks are targeted towards front-end development, with a focus on client-side rendering while 
attempting to maintain full compatibility with server-side rendering.
Some hooks have dependancies to global variables that are only available in the browser.
These hooks will have the CSR badge:

Hooks that return **components in CSR** will return *null in SRR*.  
Hooks that return **data in CSR** will return *initial or default values of the hook in SSR*.
