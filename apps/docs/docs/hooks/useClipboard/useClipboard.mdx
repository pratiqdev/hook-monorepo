## Usage

<Tabs>

<TabItem value='simple' label='Simple'>

```js 
import { useClipboard } from '@pratiq/hooks'

const [copy, value] = useClipboard()

<button onClick={() => copy('Hello!')}>Hello!</button>
```

</TabItem>

<TabItem value='advanced' label='Advanced'>

```js
import { useClipboard } from '@pratiq/hooks'

const [copy, value, success] = useClipboard()

{success && <p>Copied: {value}</p>}
{items.map(item => <button onClick={() => copy(item)}>{item}</button> )}
```

</TabItem>
</Tabs>

## API 

Values used for initial state and internal settings are provided as a config
object 


### Config Example

```ts
const { copy, value, success, reset } = useClipboard()
```


### Return Values


<HeadlessTable code={2} items={[
    ['@copy','Function','The function used to copy text'],
    ['@value','string','The current value of the clipboard according to internal state'],
    ['@success','boolean','True if the text was copied to internal state and the clipboard succesfully'],
    ['@reset','Function','Reset internal state and clear the clipboard'],
]}/>


### Interfaces
```ts
export interface I_UseClipboardReturn {
    copy: Function;
    value: string;
    success: boolean;
    reset: Function;
}
```

## Hook Methods

<Card>

### copy
Copy the text argument to the clients clipboard

```ts
(text:string) => void 
```
```ts
const clip = useClipboard()

clip.copy('Some text to copy')
```
</Card>
<Card>

### reset
Reset internal state to initial values

```ts
() => void 
```
```ts
const clip = useClipboard()

clip.reset()
```

</Card>

## Internal

This hook internally uses the `useState` hook to store the current value of the clipboard
and success state.
