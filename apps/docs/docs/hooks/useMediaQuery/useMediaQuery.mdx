## Typical Use Cases

To create consistent breakpoints and responsive styles, as well as reduce repetition and
improve performance - define a set of media queries in an external file, then import where needed.



```jsx title='my-code'
import { useMediaQuery } from "@pratiq/hooks";

const mq = () => useMediaQuery({
    'sm': '(max-width: 1000px)',
    'md': '(min-width: 1001px) and (max-width: 1200px)',
    'lg': '(min-width: 1201px) and (max-width: 1400px)',
    'xl': '(min-width: 1401px)',
})

export default mq
```

## API

### Config Example

```js
const mq = useMediaQuery({
    'xl':'(min-width: 1400px)',
    'lg':'(min-width: 1200px) and (max-width: 1400px)',
    'md':'(min-width: 1000px) and (max-width: 1200px)',
    'sm':'(max-width: 1000px)',
})
```

### Interfaces

```js
type T_StringMap = { [key:string]: string }
type T_BooleanMap = { [key:string]: boolean }
type T_UseMediaQueryList = string | string[] | T_StringMap
type T_UseMediaQuerySelector = string | number | any[]

type T_UseMediaQuery = (mediaQueries: T_UseMediaQueryList) => boolean | T_UseMediaQueryGetter

type T_UseMediaQueryGetter = (selector?: T_UseMediaQuerySelector) => boolean | T_BooleanMap
```


It is best to use this hook at the top level and pass the returned object down
through props or with context to avoid unintended layout shifts when components
are mounted.
