## Usage

<Tabs>

<TabItem value='sim' label='Simple'>

```tsx live
function Cookies() {
    const [cookie, setCookie]= useCookie('STANK_BUGS')
    return (
        <div style={{ display: 'flex', gap: '.5rem'}}>
            <code>{cookie}</code>
            <Button onClick={() => setCookie('Halyomorpha halys')}>Stinky Boi</Button>
            <Button onClick={() => setCookie('Arilus cristatus')}>Angry Boi</Button>
            <Button onClick={() => setCookie('Necrophila americana')}>Danger Boi</Button>
            <Button onClick={() => setCookie('Staphylinus caesareus')}>Hairy Boi</Button>
        </>
    )
}
```

</TabItem>

<TabItem value='adv' label='Advanced'>

```tsx live
function Cookies() {
    const [cookie, setCookie, removeCookie]= useCookie('COOL_FRUITS')
    return (
        <div style={{ display: 'flex', gap: '.5rem'}}>
            <code>{cookie}</code>
            <Button onClick={() => setCookie('Nectarine')}>Nectarine</Button>
            <Button onClick={() => setCookie('Tangerine')}>Tangerine</Button>
            <Button onClick={() => setCookie('Dragonfruit')}>Dragonfruit</Button>
            <Button onClick={() => setCookie('Elderberry')}>Elderberry</Button>
            <Button onClick={removeCookie}>Remove Cookie</Button>
        </>
    )
}
```

</TabItem>

## API 


### Interfaces
```ts
export interface I_UseCookieConfig {
    key: string;
    initialValue?: string;
}
export interface I_UseCookieReturn {
    cookie: string;
    setCookie: Function;
    removeCookie: Function;
}
```


### Config Example

```ts
const [cookie, setCookie, removeCookie] = useCookie('KEY', 'Hello, World!')
```


### Return Values

## Hook Methods


<Card>

### setCookie
Set a cookie and store the current state.

```ts
(value:string) => void 
```
```ts
const [cookie, setCookie] = useCookie('MY_KEY')

setCookie('Save this cookie @ "MY_KEY"')
```

</Card>
<Card>

### removeCookie
Delete a cookie and empty the current state.

```ts
(value:string) => void 
```
```ts
const [cookie, setCookie, removeCookie] = useCookie('MY_KEY')

removeCookie()
```

</Card>

## Internal

<StatusBadge status={1} />

This hook retrieves the value of a cookie, by the provided key, and sets the internal state of the 
`cookie` value, or the `defaultValue` if provided. The `setCookie` function sets the value of the
browser cookie, then uses the current value of the browser cookie to update the internal state.
