## Usage


<Tabs>

<TabItem value='sim' label="Simple">

```jsx live 
function ClampedCounter(props) {
  const { container, button, row, column, display } = styles.useClamp
  
  const [value, setValue, reset] = useClamp({
    value: 0,
    min: 1,
    max: 10
  })

  return (
    <div style={container}>
    <div style={row}>
        <button onClick={() => setValue(n => n - 1)} style={button}>-</button>
        <p style={display}>{value}</p>
        <button onClick={() => setValue(n => n + 1)} style={button}>+</button>
    </div>
    </div>
  );
}
``` 

</TabItem>

<TabItem value='adv' label="Advanced">

```jsx live 
function ClampedCounter(props) {
  const { container, button, row, column, display } = styles.useClamp
  
  const [value, setValue, reset] = useClamp({
    value: 0,
    min: 0,
    max: 10
  })

  return (
    <div style={container}>
        <div style={row}>
            <button onClick={() => setValue(n => n - 1)} style={button}>-</button>
            <p style={display}>{value}</p>
            <button onClick={() => setValue(n => n + 1)} style={button}>+</button>
        </div>
        <div style={column}>
            <button style={button} onClick={() => setValue(10)}>Set 10</button>
            <button style={button} onClick={() => setValue(0)}>Set 0</button>
            <button style={button} onClick={reset}>Reset</button>
        </div>
    </div>
  );
}
```

## API 

Values used for initial state and internal settings are provided as a config
object 


### Config Example

```ts
const <I_UseClampReturn> = useClamp(<I_UseClampConfig>)
```


### Return Values

<HeadlessTable code={2} items={[
    ['@value','number', 'The current state of the clamped value'],
    ['@setValue','Function', 'Set a new value, will be rejected if any type other than number and clamped within the provided range'],
    ['@reset','Function','Reset the internal state to the initial values',],
]}/>


### Interfaces
```ts
export interface I_UseClampConfig { // example
    min?: number;                   // 0
    max?: number;                   // 99
    value?: number;                 // 7
}

export interface I_UseClampReturn { // example
    value: number;                  // 7
    setValue: Function;             // setValue(9)
    reset: Function;                // reset()
}
```

## Hook Methods

<Card>

### setValue
Set the current state. Will be clamped

```ts
(newState) => void 
```
```ts
const arr = useStateArray([1, 2, 3])

arr.reset()  // [1, 2, 3]
```

</Card>


<Card>

### reset
ResetArray the array to the initial value.

```ts
() => void 
```
```ts
const arr = useStateArray([1, 2, 3])

arr.reset()  // [1, 2, 3]
```

</Card>

## Internal

This hook internally uses the `useState` hook to store the array and provides 
a wrapper around common array modification methods. 

Methods in `useStateArray` do not provide access to the current state for callback 
functions.
